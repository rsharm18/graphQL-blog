# graphQL-blog
# Pre-Requisites
### The below apps must be installed in your system before you try to run the app
1. NodeJS
2. Angular
3. NestJS
4. Typescript

## Start the service
from the roor folder navigate to api folder (cd api) and run
```sh
- npm i
- npm run serve
```

# Launch the app
from the root folder naviagte to app (cd app) and run
```sh
 - npm i
 - ng serve
 ```
